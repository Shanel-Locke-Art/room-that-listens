THE ROOM THAT LISTENS
A tiny poem-game made for itch.io.

Controls:
- Move: WASD / Arrow keys
- Interact: E
- Pause: Esc

How to upload to itch.io:
1) Zip index.html, style.css, game.js, README.txt into one .zip
2) On itch.io: Create new project
3) Kind of project: HTML
4) Upload the .zip
5) Check: "This file will be played in the browser"
6) Recommended: set viewport to 960x540 (or leave default)
